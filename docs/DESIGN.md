# Design Documentation for Wedding Decor Website

## Color Scheme
- **Primary Color**: #FF6F61 (Coral)
- **Secondary Color**: #6F61FF (Purple)
- **Accent Color**: #61FF6F (Green)

## Typography
- **Headings**: 'Georgia', serif
- **Body Text**: 'Arial', sans-serif
- **Button Text**: 'Helvetica', sans-serif

## Component Library
1. **Button**  
   - Styles: Rounded corners, shadow effect, hover animation.
   - Colors: Primary color for default, secondary color for hover.

2. **Card**  
   - Styles: Drop shadow, subtle hover effect.
   - Usage: For displaying decoration items.

3. **Navbar**  
   - Styles: Fixed position, transparent with a smooth background on scroll.

## Wireframe Descriptions
- **Home Page**:  
  - A large hero image with a call-to-action button.  
  - Highlighted featured decor items below.

- **Gallery Page**:  
  - A grid layout displaying images of different wedding decor themes.

- **Contact Page**:  
  - A simple form for inquiries along with a map location.
