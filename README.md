# Shadishine

## Project Description

Shadishine is a web application aimed at providing a streamlined experience for users. This repository contains both the frontend and backend parts of the application along with documentation.

## Setup Instructions

1. Clone the repository
2. Navigate to the project directory
3. Run `npm install` to install dependencies
4. Start the servers

### Frontend
- Navigate to `frontend` directory
- Open `index.html` in your web browser

### Backend
- Navigate to `backend` directory
- Run `node server.js`
